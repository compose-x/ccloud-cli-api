============
Installation
============


Stable release
--------------

To install ccloud-cli-api, run this command in your terminal:

.. code-block:: console

    $ pip install ccloud_cli_api

This is the preferred method to install ccloud-cli-api, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for ccloud-cli-api can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/compose-x/ccloud_cli_api

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/compose-x/ccloud_cli_api/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/compose-x/ccloud_cli_api
.. _tarball: https://github.com/compose-x/ccloud_cli_api/tarball/master
